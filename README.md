# 📊 My Data Project (MDP)

## Overview
This project demonstrates my ability to perform **data analytics and business insights** using real-world datasets.  
It covers **sales performance analysis** of beauty products (2025–2026) and **student exam results** (2026), showcasing skills in data cleaning, exploratory analysis, visualization, and reporting.

---

## 🔧 Tools & Technologies
- **Python** (Pandas, NumPy, Matplotlib, Seaborn)
- **Excel** (pivot tables, charts, formulas)
- **SQL** (basic queries for structured data)
- **GitHub** (version control, collaboration)

---

## 📈 Beauty Products Sales Analysis (2025–2026)
- **Total Sales Growth:** $10,840 → $17,320 (≈60% increase)
- **Net Profit Growth:** $350 → $2,800 (≈700% increase)
- **Star Product:** Cetaphil Moisturizer (400% sales growth, turned losses into profit)
- **Challenges:** Sebamed Lotion & Shampoo consistently loss-making due to returns
- **Stable Performers:** Sunscreen, Lip Balm, Cetaphil Lotion

---

## 🎓 Students Exam Results (2026)
- **Top Scorers:** Sasi (93%), Saski (≈97–98 across subjects)
- **Consistent Performers:** Keerthiga, Ramu, Somu (80%+)
- **Weak Performers:** Mark (389 marks), Asya (241 marks)
- **Insights:** Strong science scores from Somu & Sasi; English/Tamil weaker for Mark
